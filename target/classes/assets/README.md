Put map builder images in these folders:

- `assets/terrain`
- `assets/structures`

File naming convention (lowercase kebab-case `.png`):

Terrain files:
- `plains.png`
- `forest.png`
- `mountain.png`
- `water.png`
- `archipelago.png`

Structure files:
- `command-centre.png`
- `oil-rig.png`
- `war-factory.png`
- `air-control.png`
- `sea-control.png`
